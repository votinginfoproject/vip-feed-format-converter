# vip-feed-format-converter

Converts VIP feeds from one format to another.

Currently only supports 5.1 XML -> 5.1 CSV.

## Usage

    $ lein uberjar
    $ mkdir csv-out-dir
    $ java -jar target/vip-feed-format-converter-0.1.0-standalone.jar input.xml csv-out-dir
